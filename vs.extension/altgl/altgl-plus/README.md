# AltGr plus
AltGr plus is a snippet extention for visual studio code
## Snippet shortcut
ctrl+alt+ f   = for loop
ctrl+alt+ i   = if
ctrl+alt+ l   = console.log
ctrl+alt+ {   = function
ctrl+alt+ >   = arrow => { }
ctrl+alt+ w   = while loop 
ctrl+alt+ $   = ${}
ctrl+alt+ :   = object name : "value"
ctrl+alt+ s   = switch
ctrl+alt+ t   = try catch
